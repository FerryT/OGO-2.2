Cubes == { x : powerset natnum | forall y : x | exists z : natnum | y^3 = z }

.-- Median ---------------------------------------------------------------
|
|  s? : powerset natnum
|  m! : natnum
|
+-------------------------------------------------------------------------
|
| exists a,b : powerset s? [ a union b = s?
|                            a intersect b = empty
|                            ( count a + count b = 0
|                              or absolute (count a + count b) = 1 )
|                            forall x : a, y : b | x < m! < y ]
|            
'-------------------------------------------------------------------------

| Input : powerset natnum

| Output : natnum
|
+-------------------------------------------------------------------------
|
| exists X : Cubes [ X subset Input
|                    all Y : Input \ X | Y notsubset Cubes ]
| Output = Median(X)
|
